# JavaScript Interview Questions

## Basics

- What is JavaScript, and how does it differ from other programming languages?
- Explain the difference between `var`, `let`, and `const`.
- What are the different data types in JavaScript?
- What is the difference between `==` and `===` operators?
- Explain the difference between `undefined` and `null`.
- What is the purpose of the `'use strict'` directive in JavaScript?
- How does hoisting work in JavaScript?
- Explain the difference between function declaration and function expression.
- What is the difference between synchronous and asynchronous code?
- What are the different types of loops in JavaScript?

## Intermediate

- What are template literals, and how are they useful?
- What is destructuring in JavaScript? Provide an example.
- What is the difference between spread and rest operators?
- Explain higher-order functions with an example.
- What is closure in JavaScript? Provide a use case.
- How does the event loop work in JavaScript?
- Explain promises in JavaScript. How do `async/await` work?
- What is the difference between deep copy and shallow copy in JavaScript?
- Explain how JavaScript handles memory management and garbage collection.
- How does `localStorage` differ from `sessionStorage` and cookies?

## Advanced

- What is prototypal inheritance, and how does it work?
- Explain the `this` keyword in JavaScript and how it behaves in different contexts.
- What is the difference between `call`, `apply`, and `bind` methods?
- What are generators and iterators in JavaScript?
- Explain how `WeakMap` and `WeakSet` work in JavaScript.
- What are WebSockets, and how do they differ from HTTP requests?
- What is functional programming? How is it used in JavaScript?
- Explain the purpose of the Proxy and Reflect APIs.
- What are Web Workers and Service Workers, and when should they be used?
- What security measures should be taken to prevent XSS and CSRF attacks in JavaScript?

## Latest ECMAScript Features

- What is optional chaining, and how does it work?
- Explain the nullish coalescing operator (`??`) with an example.
- What is `BigInt`, and why is it needed in JavaScript?
- How does top-level `await` work in JavaScript?
- What are `WeakRefs` and `FinalizationRegistry` used for?
- Explain logical assignment operators (`&&=`, `||=`, `??=`) with examples.
- What is the Temporal API, and how does it improve handling dates and times?
- What are Record & Tuple data structures, and how do they differ from objects and arrays?
